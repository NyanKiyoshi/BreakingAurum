Requirements ¶

Python ¶

For this script you will need Python2 (the minor version higher or equal to 6 (2.6+)). For that you can download it thanks to the links bellow. But if you are on a Linux distribution you probably have Python2 by default to check it just open your favorite terminal and type python2.

Else, if you don't have Python2 installed on your computer you can try to install it. Maybe these few information will help you:

Platform Link
Windows XP and up (x86 installer) Python 2.7.9
Windows XP and up (x86-64 installer) Python 2.7.9
Windows 98 and 2000 (not tested) Python 2.6.6 (or the x64)
Linux (if not already installed and not available too advanced user). On Linux (POSIX system in theory), Python2 is already available. Else, you can download the sources and compiling it.

cd /tmp && wget https://www.python.org/ftp/python/2.7.9/Python-2.7.9.tar.xz tar -xf Python-2.7.9.tar.xz && cd Python-2.7.9/ ./configure make make install # as root with `sudo` or like

The Python virtual environment requirement ¶

For this script you also need a Python2 virtual environment.
There is different ways to do that, just choose the more adapted for you.

Platform Link
With aptitude (Debian, Ubuntu, ...) sudo apt-get install python-virtualenv
On ArchLinux sudo pacman -S python2-virtualenv
With EasyInstall sudo apt-get install python-setuptools && sudo easy_install virtualenv

Python-development ¶

You also will probably need to install the development package for Python for the next step then you can try:

Platform Link
With aptitude (Debian, Ubuntu, ...) sudo apt-get install python-devel
For ArchLinux Not needed.
