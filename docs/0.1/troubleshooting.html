Troubleshooting ¶

Python egg cache directory is denied ¶

The following error occurred while trying to extract file(s) to the Python egg cache: [Error 5] Access is denied: '/home/user/.python-eggs' The Python egg cache directory is currently set to: /home/user/.python-eggs Perhaps your account does not have write access to this directory? You can change the cache directory by setting the PYTHON_EGG_CACHE environment variable to point to an accessible directory.

If the access is denied fpr the Python egg cache try to check the folder permissions or change its working directory.

  • Like that for POSIXEXPORT PYTHON_EGG_CACHE=/home/user/a_new_python-eggs
  • or like that for WindowsSET PYTHON_EGG_CACHE=C:\home\user\a_new_python-eggs

`setup.py develop` returns an error ¶

  • POSIX user ¶

    error: Setup script exited with error: command 'x86_64-linux-gnu-gcc' failed with exit status 1

    If you receive this error try to install:

    • python-dev for the using Aptitude package manager (or apt) (like Debian, Ubuntu, etc).
    • or python-devel for the user using the Yum Package Manager (CentOS, RedHat, etc.).

  • Windows user ¶

    error: Unable to find vcvarsall.bat

    If you receive this error you need VisualStudio installed on your computer (to compile a package).
    If VisualStudio is installed, just run the corresponding command in your terminal before relaunching the `setup.py develop` (the command below are just temporary and will be forgotten after closing your terminal).

    • For VisualStudio 10 (2011)SET VS90COMNTOOLS=%VS100COMNTOOLS%.
    • For VisualStudio 11 (2012)SET VS90COMNTOOLS=%VS110COMNTOOLS%.
    • For VisualStudio 12 (2013)SET VS90COMNTOOLS=%VS120COMNTOOLS%.

    If it does not work, you can try to install one of the below command by replacing .\env\Scripts\easy_install.exe by your path to the easy_install into your virtual environment.

    • For x86.\env\Scripts\easy_install.exe http://www.voidspace.org.uk/downloads/pycrypto26/pycrypto-2.6.win32-py3.2.exe.
    • Or x64.\env\Scripts\easy_install.exe http://www.voidspace.org.uk/downloads/pycrypto26/pycrypto-2.6.win-amd64-py3.2.exe.
