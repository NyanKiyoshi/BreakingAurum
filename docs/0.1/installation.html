Installation and usage ¶

Creating the Python Virtual Environment ¶

Just run virtualenv --python /usr/bin/python2 --no-site-packages env.

Get the project ¶

Now, we need the paste's files to install and use it. This guide will be only for POSIX users, then if you're another user this method will probably differ.

  • You can run:

    git clone https://github.com/NyanKiyoshi/cryptopaste.git && cd cryptopaste

  • or:

    wget https://github.com/NyanKiyoshi/CryptoPaste/archive/master.zip && unzip master.zip
    cd CryptoPaste-master

Installation ¶

Now, just install the project and the others requirements: ../env/bin/python setup.py develop

If all has been perfectly done, you can (recommended) run the tests: ../env/bin/python setup.py test

If no error have been reported, just run: ../env/bin/init_db development.ini

We are now ready to make pastes!! ^-^

Running the server in development mode ¶

All is now ready for you... Just one command and the paste will be yours!

../env/bin/pserve development.ini

No error? Congratulation! You can now try to open your browser on http://127.0.0.1:6543/.
